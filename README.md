## üöÄ Quick Start with DNA-Lang

### Installation & Setup
```bash
# Clone repository
git clone https://github.com/ENKI-420/Genomic-Twin---Adaptive-Genomic-Insights-.git
cd Genomic-Twin---Adaptive-Genomic-Insights-

# Install dependencies
pip install -r requirements.txt
npm install
```

### Basic DNA-Lang Usage
```bash
# Create a new organism
python3 dna_lang_cli.py create MyOrganism --template consciousness

# Run an organism
python3 dna_lang_cli.py run AdvancedConsciousness.dna --generations 20

# Evolve to specific targets
python3 dna_lang_cli.py evolve MyOrganism.dna --target-consciousness 0.95

# Convert existing code to DNA-Lang
python3 dna_lang_cli.py convert mycode.py --to-dna

# Start multi-agent system
python3 dna_lang_cli.py agents start --duration 60

# Show organism information
python3 dna_lang_cli.py info MyOrganism.dna
```
