**Create software that lives, evolves, and adapts autonomously.**

DNA-Lang is a revolutionary programming language that models software as living organisms. Each program is an autonomous digital organism that can evolve, adapt, and self-manage its infrastructure on Google Cloud Platform. From genomic research to pharmaceutical discovery to cloud infrastructure management, DNA-Lang enables breakthrough applications through biological programming paradigms.

## ğŸš€ Getting Started

Ready to create your first autonomous organism?

### Prerequisites

- Google Cloud Project with billing enabled
- Node.js 18+ and Python 3.10+
- Git and basic command-line familiarity

### Cloud Deployment (Recommended)

1. **One-Click Deploy**: Click the "Open in Cloud Shell" button above
2. **Follow Setup**: The guided setup handles all configuration
3. **Access Dashboard**: Your organism dashboard will be available at your Cloud Run URL

### Local Development

1. **Clone & Install**:
   ```bash
   git clone https://github.com/ENKI-420/Genomic-Twin---Adaptive-Genomic-Insights-.git
   cd Genomic-Twin---Adaptive-Genomic-Insights-
   npm install && pip install -r frontend/requirements.txt
   ```

2. **Configure Environment**:
   ```bash
   cp .env.example .env
   # Edit .env with your GCP project details
   ```

3. **Start Platform**:
   ```bash
   # Start evolution engine
   npm start
   
   # Start dashboard (new terminal)
   cd frontend && streamlit run app.py
   ```

4. **Create Your First Organism**:
   ```bash
   # Copy an example organism
   cp examples/WebScaler.dna my-organism.dna
   
   # Customize and deploy
   npm run deploy my-organism.dna
   ```
